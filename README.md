## Installation
Clone the repository
```bash
git clone https://github.com/noahbjohnson/DS420.git
cd DS420
```
Install the requirements
```bash
pip install -r requirements.txt
```

## Build Documentation
```bash
cd docs
make html
open build/html/index.html
```
